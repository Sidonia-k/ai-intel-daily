# Roadmap

## Stage 0: Project Skeleton

- Create the Python package structure.
- Generate two local Markdown reports with fake data.
- Add templates, documentation, and basic tests.

## Possible Future Stages

- Add carefully selected real data sources after reviewing API terms and safety requirements.
- Add richer report sections and source attribution.
- Add scheduled generation only after the manual flow is reliable.
- Add distribution integrations only after report quality and safety policies are stable.
- Add observability and failure reporting before any production automation.

These future stages are not implemented in Stage 0.
